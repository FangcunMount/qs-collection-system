# Qlume 视觉资产清单

## 资产预算

- Banner 单文件不超过 200 KiB。
- 其他新增主包图片默认不超过 100 KiB。
- 分类小图标不超过 32 KiB。
- Banner 只承载背景和插图，标题、说明、状态、按钮必须使用 DOM。
- 新资源优先替换旧资源；`npm run check:ui-assets` 阻止未登记超限资产进入仓库。

## 生成方向记录

- 总首页：编辑式柔和插画，使用光、路径、生长意象；左侧留白；禁止文字、按钮、UI 面板和霓虹效果。
- 医学目录：克制的临床编辑插画，使用纸张、笔和细微勾选意象；禁止 3D 剪贴板、营销徽章和屏幕界面。
- 行为能力：家庭观察、积木、植物和成长路径；禁止发光大脑、机器人、神经网络和伪科技 HUD。

## 人格资产策略

人格探索现有吉祥物可以继续使用，但必须保证透明边缘干净、光源一致、显示尺寸明确；同屏只保留一个主要角色焦点。人格品牌 PNG 不进入通用 `Icon` 注册表，由领域组件直接管理。

## 登记字段

新增或替换资产必须登记：用途、领域、源文件/生成方向、版本、实际字节数、显示尺寸、替代文本、允许裁切方式、主包或分包归属。没有登记的生成初稿不得直接上线。
