# 注册模块说明

## 概述

注册功能已拆分为两个完全独立的模块：
- **用户注册** - 注册用户账号
- **受试者注册** - 注册受试者（儿童）信息

## 页面路由

### 1. 用户注册
- **路径**: `/pages/register/index`
- **用途**: 新用户注册账号
- **导航标题**: 用户注册

### 2. 受试者注册  
- **路径**: `/pages/registerChild/index`
- **用途**: 为已注册用户添加受试者信息
- **导航标题**: 受试者注册

## 使用方式

### 跳转到用户注册
```javascript
Taro.navigateTo({
  url: '/pages/register/index?goUrl=/pages/home/index'
});
```

### 跳转到受试者注册
```javascript
Taro.navigateTo({
  url: '/pages/registerChild/index?goUrl=/pages/home/index'
});
```

### 参数说明
- `goUrl`: 注册成功后跳转的页面路径
- `submitClose`: 是否在注册完成后关闭小程序（值为 "1" 时生效）

## API 接口

### 用户注册 API
```javascript
POST /users
{
  "nickname": "string",     // 用户姓名
  "avatar": "",            // 头像（可选）
  "contacts": [
    {
      "type": "phone",
      "value": "13800138000"  // 手机号
    }
  ]
}
```

### 受试者注册 API
```javascript
POST /children
{
  "name": "string",           // 受试者姓名
  "sex": "1",                 // 性别（1-男，2-女）
  "birthday": "2018-01-01"    // 出生日期
}
```

## 功能特性

### 用户注册
- ✅ 用户姓名输入
- ✅ 手机号码输入（限制11位）
- ✅ 隐私授权
- ✅ 表单验证
- ✅ 注册成功后自动跳转

### 受试者注册
- ✅ 受试者姓名输入
- ✅ 性别选择（男生/女生）
- ✅ 出生日期选择
- ✅ 表单验证
- ✅ 注册成功后自动添加到本地列表
- ✅ 自动设置为当前选中受试者

## 使用流程

### 典型场景 1: 新用户首次使用
1. 用户进入小程序
2. 跳转到 `/pages/register/index` 完成用户注册
3. 注册成功后，跳转到 `/pages/registerChild/index` 添加受试者
4. 完成后进入主页面

### 典型场景 2: 已注册用户添加受试者
1. 用户已登录
2. 直接跳转到 `/pages/registerChild/index`
3. 添加新的受试者信息
4. 完成后返回或跳转到指定页面

## 样式主题

两个注册页面共享相同的视觉风格：
- **主题色**: 紫色渐变 (#667eea → #764ba2)
- **卡片设计**: 白色圆角卡片
- **输入框**: 浅灰背景，圆角设计
- **按钮**: 渐变背景，阴影效果
